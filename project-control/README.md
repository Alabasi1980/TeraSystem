# Project Control

This folder is the project execution and traceability log.

It is not a replacement for `project-preparation/`.

- `project-preparation/` defines what should be built.
- `project-control/` records how work was assigned, executed, reviewed, accepted, blocked, deferred, or closed.

Core rule:

```text
No implementation task without a TASK-ID.
```

Tera owns project decisions.

`ProjectControlAgent`, when used, only maintains records and status updates under Tera direction.
