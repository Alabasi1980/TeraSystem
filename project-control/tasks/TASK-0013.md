## 6. Fix Details

In `app/checks/actions.ts`, in the `validateCheckData()` function, change:

```typescript
const amount = data.amount
if (amount <= 0) return { error: 'المبلغ يجب أن يكون أكبر من 0' }
```

To:

```typescript
const amount = data.amount
if (typeof amount !== 'number' || isNaN(amount) || amount <= 0) {
  return { error: 'المبلغ يجب أن يكون أكبر من 0' }
}
```

This ensures `NaN`, non-number types, zero, and negative values are all rejected.
