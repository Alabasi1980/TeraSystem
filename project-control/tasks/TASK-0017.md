## 3. Objective

Add a formal `Model Capability Gate` so Tera evaluates whether the current model is suitable before execution, balances safety/quality/cost, and asks for a stronger model only when justified.

- Update concise runtime/context/state/control references only where needed.

### Out of Scope

- No UI, route, database, migration, schema, dependency, or business-logic changes.
- No new agent generation or activation.
- No generated-agent bulk sync.
