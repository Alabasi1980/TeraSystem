## 6. Fix Details

In `validateCheckData()`, before the date comparison, add validation that both `issueDate` and `dueDate` are valid dates:

```typescript
// Validate issueDate and dueDate are valid date strings
const issueDate = new Date(data.issueDate)
const dueDate = new Date(data.dueDate)
if (isNaN(issueDate.getTime())) {
  return { error: 'تاريخ الإصدار غير صحيح' }
}
if (isNaN(dueDate.getTime())) {
  return { error: 'تاريخ الاستحقاق غير صحيح' }
}
```

Then the comparison and return values should use these validated Date objects.
