# Dynamic Dashboard Builder — بروتوتايب تمهيدي

## مصدر المعلومات
- ملف الزبون: `dynamic_dashboard_builder_blueprint.md`
- تاريخ الجلسة: 2026-07-07

## الفكرة الأساسية
تطبيق **Dynamic Dashboard Builder** — منصة ويب متعددة العملاء (Multi-Tenant) لبناء داشبوردات تفاعلية ديناميكية قابلة للربط بقواعد بيانات متعددة.

## نوع المشروع
- **النوع:** SaaS / Multi-Tenant Dashboard Platform
- **المرحلة الحالية:** Prototype (بروتوتايب تمهيدي لعرضه على الزبون)
- **الهدف:** إقناع الزبون بقدرتنا على تنفيذ المشروع الكامل

## نطاق البروتوتايب (فقط)
البروتوتايب يغطي **الجانب البصري فقط** — لا يوجد باك إند حقيقي، ولا قواعد بيانات، ولا استعلامات. الهدف هو إظهار:
1. جمالية التصميم و الفخامة البصرية
2. تدفق الشاشات الرئيسية
3. التفاعلات والحركات
4. بطاقات KPI والرسوم البيانية
5. الـ DrillDown (نقرة ← Modal)

## الجمهور المستهدف للبروتوتايب
- صاحب القرار (الزبون) — ليس مستخدمًا تقنيًا بالضرورة
- يحتاج أن يرى "المنتج النهائي" ليقرر الاستثمار

## الشاشات المستهدفة (مقترحة)
1. شاشة تسجيل الدخول
2. شاشة اختيار العميل / الداشبورد
3. لوحة التحكم الرئيسية (Main Dashboard) — أهم شاشة
4. Modal تحليلي (DrillDown)
5. لوحة الإدارة (Admin Builder) — لمحة سريعة

## نظام الألوان (من الزبون)
- الخلفية (60%): كريمي دافئ (Warm Ivory / Cream)
- البطاقات والقوائم (30%): موكا غني (Mocha Brown)
- النصوص الأساسية: بني داكن جداً (Dark Espresso)
- الأزرار والتنبيهات (10%): نحاسي (Bronze) أو خمري داكن (Burgundy)
- ألوان البيانات: تدرجات البرتقالي الهادئ والأخضر الزيتوني
