# 07_SCREENS_AND_UI_STRUCTURE.md — شاشات البروتوتايب

## قائمة الشاشات (Prototype Scope)

### S1 — شاشة تسجيل الدخول (Login)
**الهدف:** انطباع أول فخم
**العناصر:**
- خلفية كريمي مع تدرج نحاسي خفيف
- شعار الشركة (placeholder)
- حقول: اسم المستخدم، كلمة المرور (mock)
- زر دخول نحاسي مع hover effect
- Footer: "Dynamic Dashboard Builder v1.0"
- نقشة / نمط خلفية زخرفي خفيف (pattern overlay)

### S2 — شاشة اختيار العميل / الداشبورد
**الهدف:** إظهار خاصية Multi-Tenant
**العناصر:**
- شبكة بطاقات client selection (3-4 عملاء وهميين)
- كل بطاقة: اسم العميل، أيقونة، لون مميز
- زر "استعراض" لكل عميل
- Header بخلفية كريمي واسم المستخدم

### S3 — لوحة التحكم الرئيسية (Main Dashboard) ⭐
**الهدف:** الشاشة الأهم — تظهر كل قوة التطبيق
**العناصر:**
- **الشريط الجانبي (Sidebar):** داكن (espresso)، عناوين، أيقونات، شعار، زر logout
- **الشريط العلوي (Header):** خبز Breadcrumbs، أزرار إشعارات، صورة مستخدم
- **شريط الفلاتر:** Date Range, Branch, Status — مع زر "تحديث"
- **بطاقات KPI (4-6):**
  - إجمالي المبيعات (Copper)
  - صافي الإيراد (Olive)
  - العملاء الجدد (Soft Orange)
  - الطلبات النشطة (Burgundy)
  - نسبة الإنجاز (Golden Olive)
  - المؤشرات مع نسبة تغير ↑↓
- **الرسوم البيانية (2-3):**
  - رسم خطي: المبيعات خلال 12 شهر (Copper)
  - رسم دائري (Donut): توزيع المبيعات حسب الفئة
  - رسم أعمدة: المقارنة الشهرية
- **جدول سريع:** آخر 5 معاملات أو طلبات

### S4 — Modal تحليلي (DrillDown)
**الهدف:** إظهار التفاعل والتحليل عند النقر
**المحفز:** النقر على بطاقة KPI
**العناصر:**
- Modal كبير بعنوان البطاقة
- تبويبان: "جدول تفصيلي" و "تحليل"
- التبويب الأول: جدول بأعمدة (الفرع، القيمة، النسبة، الحالة)
- التبويب الثاني: رسم بياني توضيحي
- زر "تفاصيل أوسع" — يوسع المودال
- تأثير Framer Motion smooth

### S5 — Admin Builder (لمحة)
**الهدف:** إظهار قوة المنصة (لوحة الإدارة)
**العناصر:**
- قائمة جانبية (admin links)
- نموذج إعدادات بسيط
- بطاقات إدارة: العملاء، المستخدمين، الداشبوردات، مصادر البيانات
- طابع "تحت الإنشاء" للعناصر المعقدة

---

## تدفق التنقل (User Flow)

```
Login (S1)
  → Client Select (S2)
    → Main Dashboard (S3)
      → [نقر على بطاقة] → Modal تحليلي (S4)
      → [نقر على Admin] → Admin Builder (S5)
```
