# لون الزبون — المصدر الخام

## الألوان المطلوبة (نص الزبون)

> لوحة الألوان النحاسي والخمري (الذوق الكلاسيكي الراقي)
> تعتبر هذه اللوحة ممتازة للمشاريع التجارية والبيانات التي تعكس الذوق الرفيع والحرفية.
>
> - الخلفية (60%): لون كريمي دافئ (Warm Ivory / Cream)
> - البطاقات والقوائم (30%): لون موكا غني (Mocha Brown)
> - النصوص الأساسية: بني داكن جداً (Dark Espresso)
> - الأزرار والتنبيهات (10%): لون نحاسي (Bronze) أو خمري داكن (Burgundy)
> - ألوان البيانات: (تدرجات اللون البرتقالي الهادئ والأخضر الزيتوني)
