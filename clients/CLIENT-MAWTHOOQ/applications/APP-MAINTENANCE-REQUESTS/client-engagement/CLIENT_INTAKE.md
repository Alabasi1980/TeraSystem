> **نسخة محدّثة:** 2026-07-05 — توضيح نطاق المراحل حسب قرار Majed (قطع الغيار → Phase 2).

|-------|--------|

### الأجهزة التي تشملها الصيانة
- غسالات
- ثلاجات
- مكيفات
- شاشات
- أفران
- أجهزة مطابخ
- أجهزة إلكترونية خفيفة

## 2. المشكلة والحل

### المشكلة الحالية
- العمل يعتمد على دفتر يدوي + Excel + واتساب
- لا يوجد رقم موحد لكل طلب
- حالات الطلب غير موحدة
- متابعة الفنيين صعبة
- بيانات العملاء مكررة وغير منظمة
- لا يعرفون هل وافق العميل على السعر أم لا
- لا توجد تقارير يومية/شهرية
- لا توجد تنبيهات للطلبات المتأخرة

### الحل المطلوب
نظام إدارة طلبات الصيانة (Work Order Management System):
- إدارة العملاء (سجل موحد)
- إدارة طلبات الصيانة (دورة كاملة)
- حالات موحدة للطلبات
- إدارة الفنيين والصلاحيات
- تسجيل التكلفة والدفع
- تقارير أساسية

### طريقة العمل الحالية
1. العميل يتصل أو يرسل واتساب
2. الموظف يسجل الاسم والرقم ونوع الجهاز والمشكلة
3. تحديد: فني يذهب للمنزل أو العميل يحضر للورشة
4. الفني يفحص الجهاز
5. إبلاغ العميل بالتكلفة
6. موافقة العميل ← بدء الصيانة
7. تسجيل الدفع وتسليم الجهاز / إغلاق الطلب

## 3. المستخدمون المتوقعون

| المستخدم | العدد | الصلاحيات المتوقعة |
|----------|-------|-------------------|
| المدير / صاحب المؤسسة | 1 | صلاحية كاملة |
| موظف استقبال الطلبات | 1 | تسجيل وإدارة العملاء والطلبات |
| فني صيانة | 4 | عرض طلباته، تحديث الحالة، تسجيل التكلفة |
| محاسب / مسؤول مالي | 1 | إدارة المدفوعات والتقارير المالية |

## 4. النطاق المبدئي (بعد التحديث)

### المرحلة الأولى (MVP) — النطاق الأساسي
- إدارة العملاء (تسجيل، بحث، سجل)
- إدارة طلبات الصيانة (إنشاء، متابعة، حالات)
- حالات الطلب الموحدة (10 حالات)
- إدارة الفنيين وتعيينهم على الطلبات
- صلاحيات أساسية (مدير، استقبال، فني، محاسب)
- تسجيل التكلفة والدفع
- تقارير أساسية (يومي، شهري، متأخرات)
- ترحيل بيانات العملاء من Excel (مرة واحدة)

### المرحلة الثانية — قطع الغيار ونظام المخزون الأساسي
- تعريف قطع الغيار
- تسجيل الكميات والتكاليف
- ربط القطع بطلبات الصيانة
- إنقاص المخزون تلقائياً
- تقرير القطع المنخفضة
- تقارير متقدمة
- إشعارات واتساب/SMS (اختياري)

### المرحلة الثالثة — تحسينات وتوسعات
- تطبيق جوال للفنيين
- بوابة العميل (حساب خاص لمتابعة الطلبات)
- تحسينات حسب الاحتياج

### خارج النطاق للنسخة الأولى
- قطع الغيار والمخزون (المرحلة الثانية)
- إدارة المشتريات والموردين
- فواتير المشتريات التفصيلية
- تطبيق جوال منفصل (النسخة الأولى ويب متجاوب)
- بوابات دفع إلكتروني
- تكامل محاسبي متقدم
- واتساب API تلقائي
