## 3. Problem Statement

حالياً، العمل في المصنع يتم بشكل يدوي ومشتت:
- **الطلبات** تأتي بورق، واتساب، أو اتصالات — بدون رقم تتبع
- **حالة الطلب** غير معروفة: هل تحت الدراسة؟ في التصنيع؟ في الجودة؟ جاهز؟
- **المخططات والتعديلات** تضيع بين الورق والواتساب
- **لا توجد رؤية** عن حجم الطلبات المفتوحة أو الطاقة المشغولة
- **التكاليف** غير محسوبة بدقة لكل طلب
- **الأولويات** تتغير باستمرار دون نظام واضح
- **التصميم** يتغير أحياناً بعد بدء التصنيع دون توثيق

## 4. Solution Summary

نظام رقمي (Web-based) سحابي Safe لإدارة طلبات التصنيع والخدمات، يتضمن:

- **مسارين تشغيليين**: تصنيع جديد 🧱 / خدمة إصلاح 🔧
- **مراحل واضحة** لكل مسار مع إشعارات
- **إرفاق ملفات** (PDF, صور, Excel, DWG) مع تتبع النسخة الأخيرة
- **صلاحيات بأربعة أدوار**: مشروع / مصنع / إدارة / محاسبة
- **جدولة أولويات** يدوية لمدير المصنع
- **تسجيل مواد** منصرفة لكل طلب (بدون مخزون كامل في Phase 1)
- **لوحة تحكم** للمصنع تعرض الطلبات المفتوحة والطاقة
- **تقارير تصديرية** (Excel/PDF) للمحاسبة والإدارة
