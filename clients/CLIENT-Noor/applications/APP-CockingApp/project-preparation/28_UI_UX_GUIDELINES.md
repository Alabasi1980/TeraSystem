## 4. المسافات (Spacing)

| المقياس | القيمة | الاستخدام |
|---------|--------|-----------|
| 4px | `0.25rem` | أصغر مسافة |
| 8px | `0.5rem` | مسافة داخلية صغيرة |
| 12px | `0.75rem` | مسافة بين العناصر |
| 16px | `1rem` | مسافة افتراضية |
| 20px | `1.25rem` | مسافة متوسطة |
| 24px | `1.5rem` | مسافة بين العناصر في البطاقة |
| 32px | `2rem` | حشوة داخلية للبطاقات |
| 48px | `3rem` | مسافة داخل الأقسام الملونة |

## 5. الأزرار

> **مصدر الزوايا**: نظام Claude يستخدم 8px للأزرار (md)، 12px للبطاقات (lg)، 16px للـ Hero (xl).

| النوع | الخلفية | النص | الحواف | الظل | hover |
|-------|---------|------|--------|------|-------|
| **Primary** | `#cc785c` | `#ffffff` | `8px` | لا — العمق من اللون | `#a9583e` (أغمق) |
| **Secondary** | شفاف / حد `#e6dfd8` | `#141413` | `8px` | لا | خلفية `#f5f0e8` |
| **Secondary on Dark** | `#252320` | `#faf9f5` | `8px` | لا | خلفية أفتح |
| **Ghost** | شفاف | `#cc785c` | `8px` | لا | خلفية `#f5f0e8` |
| **Danger** | `#c64545` | `#ffffff` | `8px` | لا | لون أغمق |
| **Icon** | شفاف | — | `8px` (أو دائري 36px) | لا | خلفية `#f5f0e8` |

### مواصفات الأزرار

```css
.btn-primary {
  background-color: #cc785c;
  color: #ffffff;
  border-radius: 8px;
  padding: 10px 20px;
  height: 40px;
  font-weight: 500;
  font-size: 14px;
  border: none;
  cursor: pointer;
  transition: background-color 0.2s;
}
.btn-primary:hover {
  background-color: #a9583e;
}
.btn-primary:disabled {
  background-color: #e6dfd8;
  cursor: not-allowed;
}
```

## 6. البطاقات (Cards)

> **مبدأ نظام Claude**: العمق البصري يأتي من تدرجات الكريمة، **ليس من الظلال**. لا استخدام للظلال.

### أنواع البطاقات

| العنصر | المواصفات |
|--------|-----------|
| **بطاقة محتوى (Feature Card)** | خلفية `#efe9de`، زوايا `12px`، حشوة `32px` |
| **بطاقة وصفة (Recipe Card)** | خلفية `#efe9de`، زوايا `12px`، حشوة `24px` |
| **بطاقة داكنة (Dark Card)** | خلفية `#181715`، زوايا `12px`، حشوة `32px` |
| **بطاقة مع حد (Bordered Card)** | خلفية `#faf9f5`، حد `1px solid #e6dfd8`، زوايا `12px`، حشوة `32px` |
| **بطاقة اتصال (Connector Tile)** | خلفية `#efe9de`، زوايا `12px`، حشوة `24px` |
| **بطاقة كود (Code Window)** | خلفية `#1f1e1b`، زوايا `12px`، حشوة `24px` |

### Recipe Card — مواصفات

```
┌────────────────────────┐
│  ┌──────────────────┐  │
│  │    صورة الوصفة     │  │
│  │    (زوايا 8px)    │  │
│  └──────────────────┘  │
│  عنوان الوصفة           │
│  ⏱️ 30 دقيقة  |  📐 4  │
│  [تصنيف]               │
└────────────────────────┘
```

- خلفية البطاقة: `#efe9de`
- زوايا: `12px` خارجية
- حشوة داخلية: `24px`
- **بدون ظل** — العمق من لون الكريمة نفسه
- صورة الوصفة: زوايا `8px` (md)

## 12. الـ Design Gaps

> ✅ الفجوات التي تم حلها في الإصدار الحالي من الإرشادات:
> - طابع الأسطح الكريمية وعجلة الألوان — ✅ معدلة في قسم الألوان
> - الظلال — ✅ تمت إزالتها من البطاقات والعناصر الأساسية
> - زوايا الأزرار 10px — ✅ تم توحيدها إلى 8px حسب المصدر
> - ألوان الـ hover/active/disabled للأزرار — ✅ موثقة في قسم الأزرار
> - ألوان الـ Navbar — ✅ معدلة إلى كريمية بدون ظل

> الفجوات المتبقية التي تحتاج حل أثناء التنفيذ:

| الفجوة | الحل المقترح |
|--------|-------------|
| ألوان الـ hover للروابط في النصوص | استخدام `#a9583e` (أغمق من المرجاني) |
| مكون الـ Toast/Notification | خلفية حسب الحالة مع أيقونة مناسبة، زوايا `12px` |
| مكون الـ Modal/Dialog | خلفية `#faf9f5` (كريمية)، ظل خفيف جداً، زوايا `16px` (xl)، خلفية شفافة خلفه `rgba(24,23,21,0.4)` |
| حقل البحث مع اقتراحات | قائمة منسدلة بخلفية `#faf9f5`، حد `1px solid #e6dfd8` |
| مكون الـ Skeleton/Loading | استخدام تدرج كريمي `#efe9de` ← `#f5f0e8` للحمل |
| مكون الـ Empty State | أيقونة + نص هادئ + زر CTA، كريمي فاتح |
| تحريك/أنيميشن | عدم وجود توقيتات حركة في المصدر — استخدام `0.2s` للـ hover/transition كافٍ
