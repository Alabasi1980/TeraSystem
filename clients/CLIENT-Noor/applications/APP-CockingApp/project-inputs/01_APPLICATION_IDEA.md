# Application Idea — CockingApp

> **Intake Status:** _Complete — Approved_
> **Last Updated:** 2026-06-30

---

## 1. Application Description

تطبيق ويب لعرض وصفات الطبخ مع لوحة تحكم. يسمح للـ Admin (نور) بإدارة الوصفات والتصنيفات والمكونات، ويعرضها للزوار بشكل عام بدون تسجيل دخول.

**المشكلة التي يحلها:** تنظيم الوصفات بشكل احترافي مع ربط المكونات (تجنب الكتابة الحرة) وإتاحة تحميل PDF وعرض صور وفيديو وتعليقات.

---

## 2. Expected Users

| الدور | الوصف |
|-------|-------|
| **Admin (نور)** | يدخل التطبيق عبر لوحة التحكم، يضيف ويعدل الوصفات والتصنيفات والمكونات |
| **زوار / Public** | يشاهدون الوصفات بدون تسجيل دخول، يمكنهم التعليق بدون حساب |

---

## 3. Main Workflows

1. **Admin يدخل لوحة التحكم** → يدير التصنيفات → يضبط المكونات (منتجات/مواد خام)
2. **Admin ينشئ وصفة** → يختار التصنيف → يضيف مقادير (مرتبطة بالمكونات) → خطوات → صور → فيديو
3. **الزائر يتصفح الوصفات** → يفتح وصفة → يشاهد التفاصيل والصور والفيديو
4. **الزائر يعلق** على الوصفة (بدون حساب)
5. **الزائر يحمل** الوصفة بصيغة PDF

---

## 4. Expected Modules or Screens

### الواجهة العامة (Public)
- الصفحة الرئيسية — عرض الوصفات مصنفة
- صفحة التصنيفات — عرض حسب الفئة
- صفحة وصفة مفصلة — مقادير، خطوات، صور، فيديو، تعليقات
- زر تحميل PDF لكل وصفة

### لوحة التحكم (Admin)
- Dashboard رئيسي
- إدارة التصنيفات (إضافة/تعديل/حذف)
- إدارة المكونات (إضافة/تعديل/حذف) — مع وحدات قياس
- إدارة الوصفات (إنشاء/تعديل/حذف)
- ربط المكونات بالوصفة مع الكمية المناسبة
- إدارة التعليقات (موافقة/حذف)
- رفع الصور والفيديو

## 5. Required Outputs

- تصفح الوصفات والفئات
- تحميل PDF لكل وصفة
- تعليقات على الوصفات

## 6. MVP Scope (Classified)

_Classified: 2026-06-30 | Protocol: MVP_DEFINITION_PROTOCOL.md | Status: Approved by Client_

### Core MVP (Phase 1A)
- لوحة تحكم Admin (Dashboard رئيسي مع إحصائيات)
- إدارة التصنيفات (إضافة / تعديل / حذف / ترتيب)
- إدارة المكونات ككيانات منفصلة مع وحدات قياس (كوب، ملعقة، غرام، كغم، مل، حبة)
- إدارة الوصفات (إنشاء / تعديل مع ربط المكونات بالكميات)
- رفع الصور لكل وصفة (معرض صور)
- إضافة روابط فيديو YouTube لكل وصفة
- إضافة خطوات التحضير (نصوص منسقة)
- عرض عام للوصفات (صفحة رئيسية، تصنيفات، بحث)
- صفحة وصفة مفصلة (مقادير، خطوات، صور، فيديو)
- إدارة التعليقات من لوحة التحكم (موافقة / حذف)

### Extended MVP (Phase 1B)
- تحميل PDF للوصفات
- نظام تعليقات عامة (بدون حساب — مع موافقة مسبقة من Admin)
- 🛒 قائمة مشتريات ذكية
- 📐 مقياس الوصفات الديناميكي (Recipe Scaling)
- ⏱️ وقت التحضير والطهي + بحث / فلترة بالوقت
- ❤️ مفضلة / حفظ الوصفات (للزوار — تخزين محلي)

### Phase 2
- 📊 معلومات غذائية (سعرات حرارية، مغذيات)
- 📅 خطة وجبات أسبوعية
- 🔍 بحث بالمكونات المتوفرة
- ⭐ نظام تقييم ونجوم
- 🗑️ تقليل هدر الطعام
- 🌐 استيراد وصفات من المواقع (لصق رابط)
- 🔄 الطبخ بدون إنترنت (Offline)

### Phase 3
- 🤖 اقتراح وصفات بالذكاء الاصطناعي
- 🔄 بدائل المكونات الذكية
- 💰 ميزانية التسوق وتتبع المصروفات

### Later / Enterprise
- 📱 تطبيق جوال (Native / PWA متقدم)
- 👨‍👩‍👧‍👧 لائحة مشتريات متعاونة (مشاركة العائلة)
- 🎤 تحكم صوتي أثناء الطبخ
- 🔗 تكامل واتساب للمشاركة

### Out of Scope
- تكامل مع أجهزة المطبخ الذكية
- متجر إلكتروني لبيع المنتجات أو المكونات
- نظام اشتراكات مدفوعة للزوار
- استيراد فيديوهات من TikTok / Instagram
- نظام أدوار متعدد (Admin واحد فقط)
