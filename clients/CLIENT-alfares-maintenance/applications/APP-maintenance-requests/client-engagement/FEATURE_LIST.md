# FEATURE_LIST — نظام إدارة طلبات الصيانة الداخلي

> **تنبيه حوكمي (SCP-2026-07-04-022):** هذه القائمة تبقى **Working Draft / Not Baseline** حتى اعتماد `DISCOVERY_COVERAGE_SUMMARY.md`.

> قائمة الميزات المعتمدة لنسخة MVP الأولى.

---

## 1. الملخص التنفيذي

النسخة الأولى تركز على:
- تسجيل الدخول والصلاحيات
- إدارة طلبات الصيانة
- تحديث الحالة والتعليقات
- Dashboard بسيط
- إدارة مستخدمين أساسية

---

## 2. الميزات حسب الوحدة

### أ. المصادقة والصلاحيات

- تسجيل دخول باسم مستخدم وكلمة مرور
- تسجيل خروج
- 3 أدوار:
  - موظف
  - مسؤول صيانة
  - مدير إداري
- تقييد الوصول حسب الدور

### ب. إدارة طلبات الصيانة

- إنشاء طلب صيانة جديد
- حقول الطلب:
  - القسم/الموقع
  - نوع الصيانة
  - الأولوية
  - الوصف
- عرض الطلبات في قائمة
- تصفية حسب:
  - الحالة
  - التاريخ
  - القسم
- صفحة تفاصيل الطلب
- سجل تعليقات داخل الطلب

### ج. سير العمل

- جديد
- قيد التنفيذ
- مكتمل
- ملغي

> أي حالات إضافية مثل "بانتظار قطع/موافقة" أو "مغلق" تؤجل لمرحلة لاحقة.

### د. Dashboard الإداري

- عدد الطلبات الجديدة
- عدد الطلبات قيد التنفيذ
- عدد الطلبات المكتملة
- توزيع الطلبات حسب النوع أو الحالة إذا كان أبسط

### هـ. إدارة المستخدمين

- عرض المستخدمين
- إضافة مستخدم
- تعديل مستخدم
- تعطيل/تفعيل مستخدم

## 3. الميزات المؤجلة

- إرفاق صور/مرفقات
- إشعارات بريد إلكتروني
- تطبيق موبايل مستقل
- بحث متقدم
- تقارير PDF/Excel
- تكاملات خارجية
- تسجيل ذاتي للمستخدمين
