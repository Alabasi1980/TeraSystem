## 2. فكرة التطبيق

تطبيق **Windows Desktop (WPF/C# + SQLite)** محلي لإنشاء وإدارة عروض أسعار قطع السيارات.

### سير العمل الأساسي:
1. إنشاء عرض سعر جديد ← طباعة بدون أسعار (لإرساله للموردين)
2. استلام أسعار الموردين ← العودة للعرض وإدخال الأسعار
3. طباعة نموذج المعتمد مع التوقيعات
4. إرسال PDF عبر Outlook أو حفظه يدوياً
