# CLIENT_BRIEF — TeraQuotation

> **العميل:** يزيد ماهر
> **التطبيق:** TeraQuotation — نظام إدارة عروض أسعار قطع السيارات
> **تاريخ الإصدار:** 2026-07-13
> **حالة الوثيقة:** مسودة أولية — تنتظر اعتماد Majed

---

## 1. ملخص المشروع

تطبيق **Windows Desktop** يتيح لـ **يزيد ماهر** (فرد، صيانة آليات) إنشاء وإدارة عروض أسعار لشراء قطع السيارات باحترافية. التطبيق محلي بالكامل على جهازه، مع دعم طباعة A4، تصدير PDF، وإرسال عبر Outlook.

---

## 2. التقنية

| البند | القيمة |
|-------|:------:|

| **تكاملات** | Outlook (اختياري — COM Interop) |

---

## 3. النطاق — داخل النطاق (In Scope)

### 3.1 شاشة الدخول (Login)
- كلمة مرور للدخول

### 3.2 شاشة الإعدادات الرئيسية
- إدارة الموردين (إضافة، تعديل، حذف)
- إدارة القطع (إضافة، تعديل، حذف) مع بحث
- إدارة التوقيعات (إنشاء أسماء التوقيعات مرة واحدة)
- إعدادات الترويسة (شعار، اسم، معلومات)

### 3.3 شاشة إنشاء / تعديل عرض سعر
- تسلسل تلقائي + تاريخ + وصف
- جدول رئيسي: قطعة ← 3 موردين (لكل مورد: نوع القطعة + السعر)
- إضافة قطع من الكتالوج أو إدراج قطعة جديدة مباشرة
- حفظ العرض كمسودة (بدون أسعار) أو مكتمل (بأسعار)

### 3.4 شاشة إدارة العروض (قائمة)
- عرض جميع العروض
- بحث وتصفية
- فتح عرض للتعديل

### 3.5 طباعة وتصدير
- طباعة A4 منسقة
- تصدير PDF
- إرسال عبر Outlook (إن وُجد)

### 3.6 التقارير
1. **مقارنة أسعار الموردين** — لكل قطعة يظهر أرخص مورد
2. **أكثر القطع طلباً** — ترتيب تنازلي
3. **سجل العروض** — جميع العروض مع حالتها
4. **إجمالي قيمة العروض شهرياً**

### 3.7 النسخ الاحتياطي
- Backup تلقائي لقاعدة البيانات (محلي)

## 6. واجهات المستخدم المتوقعة (توقعي)

تقديرياً **5 شاشات رئيسية**:
1. 🔐 **شاشة الدخول** — Login
2. ⚙️ **الإعدادات** — Settings (موردين، قطع، توقيعات، ترويسة)
3. 📝 **عرض سعر جديد / تعديل** — Quotation Form
4. 📂 **قائمة العروض** — Quotation List
5. 📊 **التقارير** — Reports (شاشة واحدة مع tabs)
