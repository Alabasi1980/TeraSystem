## 6. سير العمل الأساسي

```
Login → إعدادات (مرة واحدة) ← تعريف الموردين والقطع والتوقيعات والترويسة
      ↓
إنشاء عرض سعر ← اختيار قطع من الكتالوج + 3 موردين
      ↓
حفظ ← طباعة بدون أسعار (للموردين)
      ↓
استلام الأسعار ← فتح العرض ← إدخال أسعار الموردين
      ↓
طباعة نهائية + PDF + Outlook (اختياري)
      ↓
أرشفة ← تقارير (لاحقاً)
```
