- ✅ جدول عرض يقارن 3 موردين لكل قطعة
- ✅ إدارة كتالوج القطع والموردين في شاشة إعدادات
- ✅ طباعة A4 + PDF + Outlook
- ✅ 4 تقارير تحليلية
- ✅ أرشفة العروض والرجوع إليها

## 4. Proposed Modules / Major Capabilities

يقسم التطبيق إلى **4 موديولات رئيسية**:

### موديول A: Security & Access
```
📁 Security
├── A1: Login Screen (بكلمة مرور)
└── A2: Password Management
```

### موديول B: Configuration & Reference Data
```
📁 Configuration
├── B1: Supplier Management (إضافة/تعديل/حذف)
├── B2: Item Catalog (كتالوج قطع + بحث + إضافة)
├── B3: Signature Management (إنشاء أسماء التوقيعات)
└── B4: Letterhead Setup (شعار + بيانات)
```

### موديول C: Quotation Core
```
📁 Quotations
├── C1: Create Quotation (تسلسل تلقائي + تاريخ + وصف)
├── C2: Quotation Table (قطعة + 3 موردين × نوع/سعر)
├── C3: Quick-Add Item from Quotation Screen
├── C4: Save / Edit / Archive Quotations
├── C5: Quotation List (بحث وتصفية)
├── C6: Two-Stage Printing (بدون أسعار → بأسعار)
├── C7: A4 Print + PDF Export
├── C8: Outlook Integration (اختياري)
```

### موديول D: Reports & Backup
```
📁 Reports
├── D1: Supplier Price Comparison Report
├── D2: Most Requested Items Report
├── D3: Quotation History Report
├── D4: Monthly Total Report
└── D5: Auto Database Backup
```

```
┌──────────────┐
│  🔐 Login    │  ← كلمة مرور
└──────┬───────┘
       │
       ▼
┌──────────────┐     ┌──────────────────┐
│  ⚙️ Settings │────▶│ B1: Suppliers    │

│              │     │ B3: Signatures   │
│              │     │ B4: Letterhead   │
└──────────────┘     └──────────────────┘
       │
       ▼
┌────────────────────────────────────────────┐
│          📝 Quotation Form                 │
│  C1: New Quotation (Auto-seq)              │
│  C2: Add Items from Catalog + 3 Suppliers  │
│  C3: Quick-Add New Item                    │
│  C4: Save (Draft / Complete)              │
└────────────────┬───────────────────────────┘
                 │
        ┌────────┴────────┐
        ▼                 ▼
┌───────────────┐  ┌────────────────┐

└───────┬───────┘  └────────┬───────┘
        │                   │
        ▼                   ▼
┌────────────────────────────────────────────┐
│  استلام الأسعار → فتح العرض → إدخال أسعار  │
└────────────────┬───────────────────────────┘
                 │
                 ▼
┌────────────────────────────────────────────┐
│  🖨️ طباعة نهائية + 📄 PDF + 📧 Outlook     │
└────────────────────────────────────────────┘
                 │
                 ▼
┌──────────────────┐     ┌──────────────────┐
│  📊 Reports      │     │  💾 Auto Backup  │
│  D1-D4           │     │  D5              │
└──────────────────┘     └──────────────────┘
```

### حالات عرض السعر:
```
Draft (مسودة) → Updated_with_Prices → Printed
                                     → PDF Exported
                                     → Sent_via_Outlook
```

### طباعة مرحلية:
1. **مرحلة 1:** إنشاء عرض ← طباعة **بدون أسعار** ← إرسال للموردين
2. **مرحلة 2:** استلام الأسعار ← فتح العرض ← **إدخال الأسعار**
3. **مرحلة 3:** طباعة **نهائية** + PDF + Outlook

## 9. Technical Decision Candidates

### ⚠️ **تنبيه — هذه توصيات فقط، وليست قرارات نهائية.**

| القرار | التوصية | الخيارات البديلة | الأساس المنطقي |
|:-------|:-------:|:----------------:|:--------------:|
| **لغة البرمجة** | **C# (.NET 8)** | — | متطلبات Windows + WPF + تكامل Office |
| **Framework** | **WPF (Windows Presentation Foundation)** | WinForms, Blazor Hybrid, MAUI | الأنسب لتطبيقات Desktop Windows مع طباعة دقيقة وتكامل Outlook |
| **قاعدة البيانات** | **SQLite** | SQL Server LocalDB, MS Access | خفيفة، ملف واحد، لا تحتاج تثبيت سيرفر |
| **نمط التطبيق** | **Windows Desktop** | Web App محلي (ASP.NET) | مطلوب محلي + تكامل سطح مكتب |
| **PDF Library** | **iTextSharp / IronPDF / QuestPDF** | — | لتوليد PDF احترافي |
| **Outlook Integration** | **Microsoft.Office.Interop.Outlook** | MAPI, SMTP | تكامل مباشر مع Outlook المثبت |
| **طباعة A4** | **FixedDocument / PrintDialog WPF** | RDLC Reports | تحكم مباشر بالتنسيق داخل WPF |
| **المصادقة** | **Password Hashing (SHA256/BCrypt)** | — | مستخدم واحد — بسيطة وآمنة |

### Recommendations for TeraAgent:

1. **WPF مع .NET 8** — يُفضل استخدام `CommunityToolkit.Mvvm` لتطبيق MVVM نظيف
2. **SQLite مع `Microsoft.Data.Sqlite`** أو `Entity Framework Core` حسب حجم الجهد
3. **طباعة:** استخدام `FixedDocument` في WPF للتحكم الدقيق بتنسيق A4
4. **PDF:** `QuestPDF` مفتوحة المصدر ومناسبة
5. **Outlook:** استخدام `Outlook Interop` ولكن مع معالجة حالة عدم وجود Outlook (يُعرض PDF فقط)
