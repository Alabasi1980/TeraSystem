نطاق العمل

الموضوع: نطاق العمل المعتمد — TeraQuotation

1. داخل النطاق (In Scope)

1.1 شاشة الدخول

  • حماية التطبيق بكلمة مرور
  • المستخدم: يزيد ماهر (مستخدم واحد)

1.2 شاشة الإعدادات

  • الموردين: إضافة، تعديل، حذف — قائمة ثابتة
  • القطع: كتالوج قطع مع بحث — إضافة فورية
  • التوقيعات: إنشاء أسماء التوقيعات مرة واحدة (قابلة للتعديل)
  • الترويسة: شعار العميل + اسمه + بياناته — تصميم أزرق داكن

1.3 إدارة عروض الأسعار

  • إنشاء عرض سعر بتسلسل تلقائي + تاريخ + وصف
  • جدول: قطعة + 3 موردين (لكل مورد: نوع القطعة + السعر)
  • إضافة قطع من الكتالوج أو إدراج قطعة جديدة مباشرة
  • حفظ العرض (مسودة / مكتمل)

1.4 أرشفة وإدارة العروض

  • قاعدة بيانات SQLite محلية
  • الرجوع إلى أي عرض وتعديله
  • قائمة عروض مع بحث وتصفية
  • ترقيم تلقائي

1.5 طباعة وتصدير

  • طباعة A4 مع ترويسة وتوقيعات
  • طباعة مرحلية: بدون أسعار ← بالأسعار
  • تصدير PDF
  • إرسال عبر Outlook (إن وجد)

1.6 التقارير (4)

  • مقارنة أسعار الموردين
  • أكثر القطع طلباً
  • سجل العروض
  • إجمالي قيمة العروض شهرياً

1.7 النسخ الاحتياطي

  • نسخ احتياطي تلقائي لقاعدة البيانات محلياً

2. خارج النطاق (Out of Scope)

دعم مستخدمين متعددينلا حاجة

فواتير إلكترونية / ضريبيةغير مطلوبة
تكامل محاسبيغير مطلوب
استضافة سحابيةتطبيق محلي

3. الافتراضات

  • العميل سيُسلم الشعار والبيانات المطلوبة للترويسة قبل بدء التصميم
  • التطبيق يُثبّت على Windows 10 أو 11
  • Outlook غير إلزامي — يعمل بدونه (PDF فقط)
  • قاعدة البيانات SQLite محلية — لا تحتاج سيرفر

4. القيود

  • التطبيق يعمل محلياً فقط
  • واجهة المستخدم باللغة العربية
  • التصميم: أزرق داكن + تنسيق احترافي
