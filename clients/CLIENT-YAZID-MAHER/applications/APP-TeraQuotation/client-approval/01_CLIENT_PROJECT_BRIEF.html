فكرة التطبيق

تطبيق Windows Desktop يتيح إنشاء وإدارة عروض أسعار لشراء قطع السيارات باحترافية. يقوم العميل بإنشاء عرض سعر يحدد فيه القطع المطلوب شراؤها، مع إضافة أسعار 3 موردين مختلفين، ثم طباعة النموذج أو تصديره كـ PDF أو إرساله عبر Outlook.

المشكلة التي يحلها

  • تنظيم عملية شراء قطع السيارات بعروض أسعار موحدة
  • مقارنة أسعار 3 موردين في نموذج واحد
  • طباعة مرحلية (بدون أسعار ← للموردين، ثم بالأسعار ← للاعتماد)
  • أرشفة العروض السابقة والرجوع إليها

الهدف من التطبيق

تقديم أداة احترافية ومبسطة لإدارة عروض الأسعار، توفر وقت العميل وتنظم مشترياته.
