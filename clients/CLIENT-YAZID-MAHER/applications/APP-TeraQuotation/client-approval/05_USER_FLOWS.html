التدفق 1: إنشاء عرض سعر جديد

الدخول إلى التطبيق (كلمة المرور) │ ▼ الذهاب إلى "عرض سعر جديد" │ ▼ إدخال المعلومات الأساسية: وصف، تاريخ (تلقائي)، رقم تسلسلي (تلقائي) │ ▼ إضافة البنود: اختيار قطعة ← بحث ← أو إضافة قطعة جديدة ← إدخال النوع والسعر لكل مورد (3 موردين) │ ▼ حفظ العرض (كمسودة أو مكتمل) │ ▼ طباعة / PDF / Outlook

التدفق 2: طباعة مرحلية

إنشاء عرض سعر بدون أسعار │ ▼ طباعة نسخة "للموردين" (بدون أسعار) ← إرسالها يدوياً │ ▼ استلام الأسعار ← فتح العرض المحفوظ ← إدخال الأسعار │ ▼ طباعة النسخة النهائية (بأسعار) ← PDF ← Outlook للاعتماد

التدفق 3: إدارة العروض السابقة

الدخول إلى "قائمة العروض" ← بحث (بالتاريخ/الرقم/الوصف) │ ▼ فتح العرض ← عرض التفاصيل │ ├── تعديل (إضافة/تعديل بنود) └── طباعة / تصدير / إرسال

التدفق 4: إدارة الإعدادات

الدخول إلى "الإعدادات" │ ▼ الموردين ← إضافة / تعديل / حذف القطع ← إضافة / تعديل / حذف / بحث التوقيعات ← إنشاء أسماء التوقيعات الترويسة ← تحميل الشعار + إدخال البيانات
