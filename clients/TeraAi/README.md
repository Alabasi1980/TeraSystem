# TeraOpenCode

TeraOpenCode is a standalone AI coding engine, forked from [OpenCode](https://github.com/anomalyco/opencode) and owned by **TeraSystem**.

This is a fully independent fork. It is not affiliated with, nor receives updates from, the upstream OpenCode project.

## What is TeraOpenCode?

TeraOpenCode is the execution engine behind TeraSystem — an operating system for client and software project delivery. It provides:

- A terminal-based AI coding agent (TUI)
- A desktop application
- A web console
- A plugin and SDK system for deep customization
- Session-based durable AI conversation history

## Status

This project is under active transformation. TeraSystem is progressively integrating its delivery methodology, policies, and project context directly into the engine runtime.
