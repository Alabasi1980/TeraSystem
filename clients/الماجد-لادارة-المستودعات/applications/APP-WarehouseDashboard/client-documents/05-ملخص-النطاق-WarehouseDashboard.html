السادة المحترمون،

تحية طيبة وبعد،

في هذا الموجز عرض شامل لنطاق المرحلة الأولى (MVP) من مشروع WarehouseDashboard، يشمل الميزات الرئيسية والمكونات الفرعية والبنود المؤجلة والمستبعدة.

أوّلاً: الميزات الأساسية (Phase 1 — 9 ميزات رئيسية)

API Application — WarehouseDashboard.API (4 ميزات)

  • API-01 — الاتصال بقاعدة Oracle واستخراج البيانات: تكوين اتصال ODP.NET، اكتشاف الجداول، تحويل أنواع البيانات، معالجة الأخطاء.
  • API-02 — تحميل البيانات إلى SQL Server: حذف الموجود + إدخال مجمّع باستخدام SqlBulkCopy، إدارة المعاملات، ترتيب الجداول حسب التبعيات.
  • API-03 — محرك المزامنة: مزامنة تلقائية (Background Service) ويدوية (API Endpoint)، عرض حالة المزامنة، إعدادات قابلة للتكوين.
  • API-04 — سجلات المزامنة: تسجيل كل عملية (وقت، حالة، عدد سجلات، أخطاء)، حفظ في SQL Server، API لجلب السجلات.

Razor Dashboard — WarehouseDashboard.Web (5 ميزات)

  • RZR-01 — تخطيط Dashboard وعرض البطاقات: تخطيط شبكي متجاوب مع ~20 بطاقة ديناميكية، استخدام Syncfusion، تطبيق الهوية البصرية (Blue Theme).
  • RZR-02 — خاصية Drill Down: تعمق عبر مستويات قابلة للتكوين لكل بطاقة، شريط تنقل (Breadcrumb)، رجوع للمستوى السابق.
  • RZR-03 — ربط البيانات وتنفيذ الاستعلامات: تنفيذ SQL Queries و Views من التكوين، ربط النتائج بأنواع البطاقات.
  • RZR-04 — حالة المزامنة وأدوات التحكم: عرض آخر وقت مزامنة، مؤشر الحالة، زر مزامنة يدوية، عرض سجلات المزامنة.
  • RZR-05 — Admin Panel: حماية بكلمة مرور، وصول عبر الرابط المباشر، CRUD للبطاقات، تكوين مصادر البيانات، اختيار أنواع العرض، تكوين Drill Down، اختبار الاستعلامات.

البنية التحتية (3 ميزات)

  • INF-01 — هيكل المشروع: حل .NET 8 يحتوي على مشروعين (API + Razor) مع مكتبة مشتركة اختيارية.
  • INF-02 — إعداد قاعدة البيانات: جداول التكوين، جداول السجلات، جداول البيانات المنقولة (تُحدد أثناء التنفيذ).
  • INF-03 — النشر على IIS: إعداد IIS، إنشاء Application Pool، تكوين المسارات.

إجمالي Phase 1: 9 ميزات رئيسية تحتوي على 33 مكوناً فرعياً (جميعها P1 — Must-have).

ثانياً: المرحلة الثانية (Phase 2 — مؤجلة)

  • شاشات تعديل البيانات: إضافة وتعديل وحذف بيانات في SQL Server عبر واجهة مستخدم.
  • صلاحيات المستخدمين: نظام تسجيل دخول متعدد المستخدمين مع صلاحيات مختلفة.
  • التصدير إلى Excel/PDF: تصدير البيانات والتقارير من Dashboard.
  • تحليلات متقدمة: تقارير متقدمة وتحليلات إضافية.

رابعاً: القرارات التقنية الرئيسية

  • .NET 8 (LTS) — الإطار الرئيسي للتطوير
  • ASP.NET Core Razor Pages — لنمط الواجهة
  • Syncfusion مع Community License — للداشبورد والبطاقات
  • ODP.NET (Oracle.ManagedDataAccess) — للاتصال بقاعدة Oracle
  • ADO.NET SqlBulkCopy للإدخال المجمّع + EF Core للتكوين فقط
  • Full Refresh مع دعم Incremental Sync جاهزية استراتيجية
  • النشر على IIS محلي — داخل شبكة الشركة
  • الهوية البصرية: Blue Theme (11 لوناً محدّداً)
