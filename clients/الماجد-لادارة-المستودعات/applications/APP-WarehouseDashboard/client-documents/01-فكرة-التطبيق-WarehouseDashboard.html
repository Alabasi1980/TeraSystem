الموضوع: عرض فكرة تطبيق WarehouseDashboard — نظام مزامنة البيانات ولوحة القيادة

إلى: السادة / شركة الماجد لإدارة المستودعات
المرجع: TDS-LET-2026-002

السادة المحترمون،

تحية طيبة وبعد،

يسرّ مؤسسة تيرا للنظم (Teranoo) أن تقدّم لشركتكم الموقّرة فكرة تطبيق WarehouseDashboard، وهو نظام متكامل يهدف إلى مزامنة البيانات من قاعدة Oracle وعرضها في لوحة قيادة (Dashboard) ديناميكية تفاعلية، بما يتيح لكم رؤية واضحة وشاملة لأداء المستودعات في الوقت الفعلي.

المشكلة التي يحلها التطبيق

تعتمد شركة الماجد لإدارة المستودعات على قاعدة بيانات Oracle تحتوي على بيانات تشغيلية مهمة، إلّا أن هذه البيانات لا تتوفّر حالياً بصورة مرئية وسهلة الاستيعاب لصانع القرار. يتطلّب استخراج البيانات وعرضها جهداً يدوياً ووقتاً، مما يؤخّر الحصول على الرؤى التشغيلية اللازمة. يأتي WarehouseDashboard ليحلّ هذه المشكلة عبر أتمتة عملية المزامنة وتقديم البيانات في Dashboard تفاعلي احترافي.

مكونات النظام

يتكوّن النظام من تطبيقين رئيسيين:

  • WarehouseDashboard.API: تطبيق خلفي (Backend) يقوم باستخراج البيانات من Oracle تلقائياً أو يدوياً، وتحويلها إلى SQL Server باستخدام تقنية الإدخال المجمّع عالي الأداء (Bulk Insert)، مع تسجيل جميع عمليات المزامنة.
  • WarehouseDashboard.Web: تطبيق واجهة أمامية (Frontend) يعرض البيانات في Dashboard ديناميكي يتسع لنحو 20 بطاقة تفاعلية قابلة للتخصيص، مع إمكانية التعمق في البيانات (Drill Down) عبر مستويات متعددة.

القدرات الرئيسية

  • مزامنة تلقائية ويدوية للبيانات من Oracle إلى SQL Server
  • لوحة قيادة ديناميكية تدعم 6 أنواع من البطاقات (مخططات بيانية، مؤشرات أداء، جداول بيانات)
  • خاصية التعمق في البيانات (Drill Down) لاكتشاف التفاصيل
  • لوحة تحكم إدارية (Admin Panel) لإدارة البطاقات وتكوينها ديناميكياً
  • تسجيل كامل لعمليات المزامنة مع إمكانية تتبّع الحالة
  • تصميم متجاوب يعمل على جميع أحجام الشاشات

التقنيات المستخدمة

  • .NET 8 (LTS) — أحدث إطار عمل من Microsoft مع دعم طويل الأمد
  • ASP.NET Core Razor Pages — لواجهة المستخدم
  • Syncfusion — مكتبة الداشبورد الاحترافية (مع رخصة مجانية متوفرة)
  • Oracle (ODP.NET) — للاتصال بقاعدة البيانات المصدر
  • SQL Server (EF Core + ADO.NET) — لقاعدة البيانات الهدف والتخزين
  • IIS — للنشر المحلي على خادم الشركة

الفوائد المتوقعة

  • الحصول على رؤية فورية وموحّدة لأداء المستودعات
  • تقليل الجهد اليدوي في استخراج البيانات وإعداد التقارير
  • تمكين صانع القرار من التعمق في البيانات لاتخاذ قرارات أسرع وأدق
  • نظام يعمل بالكامل محلياً على خوادم الشركة بمعايير أمان عالية
  • قابلية تطوير مستقبلية لإضافة ميزات إضافية مثل الصلاحيات والتقارير المتقدمة
