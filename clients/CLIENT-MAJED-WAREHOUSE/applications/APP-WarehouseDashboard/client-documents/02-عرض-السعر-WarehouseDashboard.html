مدة الصلاحية: 30 يوماً من تاريخه

السادة المحترمون،

تحية طيبة وبعد،

استناداً إلى التحليل الفني الذي أجرته مؤسسة تيرا للنظم (Teranoo) لمتطلبات شركتكم الموقّرة، نقدّم لكم عرض السعر التالي لتطوير المرحلة الأولى (MVP) من نظام WarehouseDashboard.

أوّلاً: نطاق العمل (Phase 1 — MVP)

  • WarehouseDashboard.API: تطبيق API لاستخراج البيانات من Oracle عبر ODP.NET، ومزامنتها مع SQL Server بطريقة Full Refresh (حذف + إدخال مجمّع)، مع دعم المزامنة التلقائية واليدوية وتسجيل العمليات.
  • WarehouseDashboard.Web: تطبيق Dashboard ديناميكي بحوالي 20 بطاقة تفاعلية (مخططات بيانية، مؤشرات أداء، جداول) مع خاصية Drill Down، وشريط حالة المزامنة.
  • Admin Panel: لوحة تحكم إدارية منفصلة ومحمية بكلمة مرور لإدارة البطاقات وتكوينها وتحديثها ديناميكياً.
  • البنية التحتية: إعداد حل المشروع على .NET 8، تصميم قاعدة بيانات SQL Server، ونشر النظام على IIS.

ثانياً: طريقة التسعير

Time & Material (ساعة عمل) — السعر: 4 JOD لكل ساعة. هذه الطريقة هي الأنسب للمشاريع متوسطة التعقيد حيث تكون بعض التفاصيل (مثل جداول Oracle) غير محددة بالكامل بعد وستُكتشف أثناء التنفيذ.

ثالثاً: تقدير الساعات والتكلفة

بناءً على تحليل Scorecard (مؤشر التعقيد: 45.95% — متوسط)، فإن تقدير الساعات لكل مكوّن كالتالي:

  • API (محرك المزامنة): 105 — 150 ساعة
  • Dashboard (~20 بطاقة + Drill Down): 140 — 205 ساعة
  • Admin Panel (CRUD + تكوين): 110 — 160 ساعة
  • البنية التحتية (مشروع + قاعدة بيانات + IIS): 35 — 50 ساعة
  • التكامل والاختبار: 40 — 60 ساعة

المجموع التقديري: 430 — 625 ساعة

التكلفة التقديرية الإجمالية: 1,720 — 2,500 JOD (فقط ألف وسبعمائة وعشرون إلى ألفان وخمسمائة دينار أردني)

رابعاً: شروط الدفع

  • الدفع على أساس Time & Material — يُفوتر وفقاً لساعات العمل الفعلية المسجّلة شهرياً.
  • لا يُشترط عربون مقدّم.

خامساً: المدة الزمنية

بدون مهلة زمنية محددة — العمل بالسرعة الطبيعيّة مع الالتزام بالجودة المطلوبة. المدة التقريبية: 12 — 16 أسبوعاً حسب توفر المعلومات من طرف العميل.

سادساً: البنود المستثناة (خارج هذا العرض)

  • المرحلة الثانية (Phase 2): شاشات تعديل البيانات، صلاحيات المستخدمين، التصدير إلى Excel/PDF — تُسعّر منفصلة لاحقاً.
  • الاستضافة السحابية، التطبيق الجوّال، Microservices، CI/CD، الاختبار الآلي.

سابعاً: ملاحظات

  • هذا العرض تقديري غير ملزِم حتى تاريخ التوقيع على اتفاقية الخدمة.
  • أي تغيير في النطاق بعد البدء قد يؤثّر على التكلفة والمدة.
  • مكتبة Syncfusion متوفرة برخصة مجانية (Community License — مفتاح UnlockKey موجود) ولا تتطلب تكلفة إضافية.
