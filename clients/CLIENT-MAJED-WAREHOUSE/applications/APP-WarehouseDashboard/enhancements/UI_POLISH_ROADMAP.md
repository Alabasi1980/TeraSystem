## 📐 معايير البولش لكل صفحة

عند تطوير كل صفحة، نطبق:

- [ ] **نظام الألوان** — استخدام CSS variables من Blue Identity Palette
- [ ] **Responsive** — تعمل على Desktop (1200px)، Tablet (768px)، Mobile (375px)
- [ ] **الأنيميشن** — دخول سلس (fadeInUp)، انتقالات ناعمة
- [ ] **Skeleton Loading** — موجود عند تحميل البيانات
- [ ] **Empty State** — رسالة مناسبة عند عدم وجود بيانات
- [ ] **Error State** — رسالة خطأ واضحة + زر إعادة المحاولة
- [ ] **SVG أيقونات** — بدلاً من الـ emoji
- [ ] **سرعة الأداء** — لا hover معقد، لا JS زائد
- [ ] **اتساق التصميم** — نفس الترويسة، نفس الـ border-radius، نفس الظلال
